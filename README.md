# Sistema de Gestión de Biblioteca
Sistema CRUD para gestionar libros, socios y préstamos.

## Tecnologías
- Python 3
- PostgreSQL
- psycopg2

## Archivos
- `conexion.py` — Conexión a la base de datos
- `libros.py` — Gestión de libros
- `socios.py` — Gestión de socios
- `prestamos.py` — Gestión de préstamos
- `menu.py` — Menú principal
- `schema.sql` — Estructura de la base de datos

## ▶️ Cómo ejecutarlo
1. Clonar el repositorio
   git clone https://github.com/nocedayo/sistema-biblioteca.git

2. Instalar dependencias
   pip install psycopg2-binary

3. Crear la base de datos con schema.sql en PostgreSQL

4. Editar conexion.py con tus credenciales

5. Ejecutar
   python menu.py
